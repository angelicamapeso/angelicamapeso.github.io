Simplet

App Concept

Simplet is a concept for a student calendar app, focused on progress-tracking and goal setting. The main component of the App is the “Project” feature, which is a group of “Tasks” or short-term goals that a student can set for each assignment.

As they complete these tasks and mark them off on the app, their progress bar for the assignment or project increases. In doing so, they can break down the assignment into a step-by-step plan.

Other Features:

  • Student Collaboration to assign Tasks

  • Class Schedule Sharing

  • Class Colour Coding

  • Task/Event Categories to Colour Code Similar Items

The Design

As users add on more items to a calendar app, they layout can get cluttered. To avoid this, all sections are organized by moveable tabs, which can be expanded or hidden. Additionally, the Simplet app relies heavily on colour coding as markers to represent multiple items at a time.

The Process

The original design was completed in October 2017 as part of an Introduction to Multimedia class, where rough sketches for four screens and one digital mockup was required.

It was revamped in December 2018 for a more polished look and extended to include 3 more digital mockups based on the previous rough designs.
